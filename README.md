# Collaborated Movie Diary Project

Movie Diary is a frontend web application that allows users to keep a personal journal of movies they have watched.  
Users can record thoughts, notes, and reflections about movies in a simple and structured way.

This project was built as a learning exercise to practice frontend development, project structure, and modern build tools.

## Project Structure

```
.
├── src/
│   ├── components/
│   ├── pages/
│   ├── styles/
│   └── index.js
├── .parcel-cache/
├── .postcssrc
├── package.json
├── .gitignore
└── README.md
```
## Technologies Used

- JavaScript
- HTML
- CSS
- Parcel (Bundler)
- PostCSS

## Features

- Create movie diary entries
- View journal entries
- Page-based navigation
- Basic validation for invalid pages
- Clean and simple user interface

## Scripts

Install dependencies:
npm install

Start development server:
npm start

Build for production:
npm run build

## Purpose

The purpose of this project is to:
- Practice structuring a frontend application
- Learn how to use Parcel as a bundler
- Improve JavaScript and CSS skills
- Build a functional diary-style application
